# LoseWeight
###
🏃‍♀️ 减肥健身打卡app

## 功能
* 设定打卡任务并完成，完成后勋章奖励
* 根据打卡记录生成年度、月度、周报表统计
* 查看他人饮食、运动、健康方面的动态并收藏
* 记录饮食
